Data description of the file "settlements.csv"
This file contains data on the cultural period (or site-phase) during which each site was occupied. Each row represents one site-phase at one settlement. Name (character): name of the archaeological site.Id (numeric): unique identifier for the site.StartDate (numeric): approximate start date for the cultural period expressed in absolute calendric years. EndDate (numeric): approximate end date for the cultural period expressed in absolute calendric years.Longitude (numeric): unprojected WGS84 eastingsLatitude (numeric): unprojected WGS84 northingsRain_mm (numeric): annual rainfall at the siteSource (character): publication from which the settlements data have been collected.
