Data description of the file "urban_sites.csv"
This file contains data on the cultural period (or site-phase) during which each site was occupied. Each row represents one site-phase at one settlement. Name (character): name of the archaeological site.Id (numeric): unique identifier for the site.Rainfall (numeric): annual rainfall at the site.StartDate (numeric): approximate start date for the cultural period expressed in absolute calendric years. EndDate (numeric): approximate end date for the cultural period expressed in absolute calendric years.
SizeHa (numeric): estimated site size for each cultural period.Longitude (numeric): unprojected WGS84 eastingsLatitude (numeric): unprojected WGS84 northingsSource (character): publication from which the urban settlement data have been collected.
